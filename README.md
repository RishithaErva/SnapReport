# SnapReport by Snaphomz

SnapReport generates a branded AI-written real estate market report PDF for any ZIP code in under 90 seconds.

## Setup

```bash
pip install -r requirements.txt
```

## Set your API key

```bash
export GOOGLE_API_KEY="your-key-here"
```

## Run

```bash
streamlit run app.py
```
